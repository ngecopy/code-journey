Library kode

Library kode yang sudah dipelajari selama ini.

1

!DOCTYPE html

Digunakan untuk mendeklarasikan bahwa kode tersebut adalah html

2

html

Digunakan untuk pembukaan kode html.

3

head

Digunakan untuk menaruh informasi mengenai html yang dibuat.

4

body

Digunakan untuk membuat kerangka yang akan ditampilkan.
